# White Search / 白搜 （百度搜索优化、聚合搜索）

起因是从2026年某天百度首页增加了一个无法手动关闭的信息流
固诞生白搜，可作为百度的去广告增强版用于浏览器首页，提供百度、Bing、Google 多引擎的一键搜索直达，并集成（保留）联想词建议以及常见AI Chat入口，整体采用玻璃风格。![index](.\README\index.jpg)


## 功能

- 默认回车跳转百度搜索结果
- Baidu、Bing、Google 一键重定向
- 补充多引擎入口和更现代的界面
- 百度联想词建议，支持键盘上下选择
- 每次进入焕然一新的渐变背景
- 液态玻璃风格界面


## 说明

- 项目为纯静态页面，无需安装依赖或启动服务
